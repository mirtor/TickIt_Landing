FAQ

Short answers for the most common questions.

Can this landing install the app for me?

Not directly. Browsers don’t allow one website to trigger installation of a different-origin PWA. This page can guide you to do it in a few taps.

Is my data private?

Your tasks are scoped by your account in Firestore. The project owner can technically access database content unless you implement end-to-end encryption. For a personal productivity app, this is a common approach.

What’s the fastest way to start?

Open the app, create your first task, then add subtasks or a bullet note.
