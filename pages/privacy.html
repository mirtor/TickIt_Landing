Política de Privacidad

Esta política explica cómo TickIt trata los datos de los usuarios. Si tienes dudas, revisa también la configuración de tu cuenta y tu navegador.

Resumen rápido: TickIt usa Firebase Authentication y Cloud Firestore para guardar tus tareas y notas asociadas a tu usuario. No vendemos datos. El acceso administrativo a la base de datos por parte del propietario del proyecto no está diseñado para analizar contenido personal.

1. Qué datos se recopilan

  • Datos de autenticación: email y/o identificadores del proveedor de login.
  • Datos de uso propios de la app: tareas, subtareas y notas creadas por ti.
  • Metadatos técnicos básicos para el funcionamiento del servicio (p. ej., timestamps).

2. Finalidad del tratamiento

Los datos se utilizan exclusivamente para:

  • Permitir el inicio de sesión y mantener tu cuenta.
  • Sincronizar tus tareas y notas entre dispositivos.
  • Mejorar la estabilidad y experiencia de la app.

3. Base tecnológica

TickIt está construido con Vue y utiliza servicios de Google Firebase:

  • Firebase Authentication para la gestión de usuarios.
  • Cloud Firestore para almacenar datos de tareas y notas.
  • App Check (reCAPTCHA Enterprise) para proteger el uso del backend frente a abuso.

4. Dónde se almacenan los datos

Los datos se almacenan en la infraestructura de Firebase/Google Cloud, bajo el proyecto asociado a TickIt. La ubicación exacta puede depender de la configuración del proyecto.

5. Compartición de datos

TickIt no comparte tus datos con terceros con fines comerciales. Los datos se procesan por Firebase como proveedor de infraestructura.

6. Seguridad

Se aplican reglas de acceso basadas en usuario para que cada persona vea únicamente sus propios documentos.

Nota importante: Esta app no implementa cifrado extremo a extremo del contenido. Técnicamente, el propietario del proyecto podría tener acceso administrativo a la base de datos. En apps personales y de productividad pequeñas, este enfoque es común. Si en el futuro se añadiera cifrado E2E, se actualizaría esta política.

7. Retención y eliminación

  • Puedes eliminar tareas y notas desde la app.
  • Si deseas eliminar tu cuenta y datos asociados, puedes solicitarlo al contacto del proyecto.

8. Cookies y tecnologías similares

Firebase y reCAPTCHA pueden utilizar cookies o almacenamiento del navegador necesarios para autenticación y seguridad. La configuración puede variar según el navegador y sus políticas de privacidad.

9. Derechos del usuario

Si te encuentras en España o en el Espacio Económico Europeo, puedes ejercer tus derechos de acceso, rectificación, supresión, limitación y portabilidad según aplique, solicitándolo al responsable del proyecto.

10. Cambios en esta política

Esta política puede actualizarse para reflejar mejoras de seguridad, cambios en la infraestructura o nuevos requisitos legales.

11. Contacto

Si necesitas soporte o quieres solicitar la eliminación de tu cuenta, contacta con el propietario del repositorio/proyecto donde se publica TickIt.
